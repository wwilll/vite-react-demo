# vite-react-demo

使用 vite 搭建的 react h5 demo，采用了最新的框架库，仅供学习交流

### 使用

```
yarn

yarn dev
```

### 开发步骤

1. 在 src/pages 中新建页面组件
2. 在 src/router/menus.tsx 中注册页面组件
3. 开始业务代码编写
4. 打包部署
